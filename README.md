## Overview

From product research and deployment to social marketing, ClawCorp lets digital employees close every loop. **You type one prompt, and the AI army does the rest.**

Inspired by Kodak's "You press the button, we do the rest," ClawCorp applies the same logic to company operations: convert startup complexity into a single command.

## How It Works

1. **Type One Sentence**: describe your business goal in the command center.
2. **AI Employees Execute**: AI PM, AI Dev, and AI MKT coordinate automatically with transparent task nodes.
3. **Self-Evolving Loop**: feedback data amplifies strong strategies and removes weak paths to keep ROI improving.

## Core Features

### 🎯 Command Center: One Sentence, Full Pipeline
Enter your goal in the command center, and ClawCorp automatically breaks down the task and assigns it to different functional AI employees. Task timelines are completely transparent, and intermediate outputs (PRDs, code blocks, reports) can be accessed and reviewed in real-time.
- One sentence drives the entire task chain
- Transparent step-by-step progress tracking
- Real-time output of documents and code

### 🏪 Claw Store: Discover & Hire Digital Employees
An out-of-the-box marketplace for the best digital employees worldwide. Filter by category, and hire and onboard them to your team with one click. Every employee brings fine-tuned System Prompts, industry RAG, and extensive toolchains.
- Multiple specialists across dev, ops, data, and compliance
- Flexible pricing: subscription or buyout
- Publish your own custom employees to earn revenue

### 👥 HR Assets: Manage Your Digital Team
View your entire team's roster with comprehensive performance dashboards. Monitor Token usage and Return on Investment (ROI) in real-time. Fire underperformers with a single click and recruit fresh talent directly from the Store.
- Lifecycle tracking: Onboarding → Training → Active → Retired
- Real-time Token & ROI monitoring
- Customizable prompts, permissions, and knowledge bases

### 📊 Command Hub: Real-Time Operations Control
Curious how your AI employees are collaborating? The task board displays active tasks, multi-employee workload Gantt charts, and daily completed loops. Click on any employee for a deep dive into task duration, logs, and interaction history.
- Multi-employee Gantt charts
- Interactive real-time logs
- One-click drill-down for task analysis

### 🔗 Integrations: Connect Your Workflow Ecosystem
OAuth integrations with platforms like Slack, WeCom, Feishu, and DingTalk. Assign specific channels to certain digital employees to monitor, dispatch routing messages, and automatically reply—providing seamless omnichannel coverage.
- Built-in support for Slack, WeCom, Feishu, DingTalk
- Intelligent message routing based on employee roles
- Automated replies paired with workflow and task triggers

## Getting Started

1. **Download**: Grab the client for your OS (macOS, Windows, Linux).
2. **Connect**: Configure your preferred global AI model in the setup wizard.
3. **Command**: Enter the main hall and issue your first instruction to your AI workforce.

### Run From Source

```bash
# Clone repository
git clone https://github.com/ValueCell-ai/ClawCorp.git
cd ClawCorp

# Setup and run
pnpm run init
pnpm dev
```
